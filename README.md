# MAG Citation Recommender

Graph Neural Network for citation recommendation using the Microsoft Academic Graph (MAG) dataset, implementing the GRIL (Graph Retrieval-Integrated Learning) framework.

## Documentation

See [GRIL_ALGORITHMS_AND_PIPELINE.md](./GRIL_ALGORITHMS_AND_PIPELINE.md) for:
- Complete pipeline overview
- Algorithm implementations
- Team responsibilities
- Milestone alignment

## Project Structure

- `notebooks_01_build_graph.ipynb` - Graph building and GNN training (Shikshya & Jacob)
- `Algorithms_Implementation.ipynb` - Algorithm 1 and GRIL implementations (Ujjwal)
- `algorithms/` - Python modules for GRIL algorithms
- `extract_components_for_algorithm1.py` - Utility to extract components from graph building notebook
